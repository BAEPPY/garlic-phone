# 갈릭폰

갈릭폰은 선생님이 방을 만들고, 학생들이 링크나 QR 코드로 접속해서 글쓰기와 그림 그리기를 번갈아 진행하는 웹 게임입니다.

## 지금 가능한 것

- 관리자 화면과 참가자 화면 분리
- 관리자는 게임에 참가하지 않고 진행만 담당
- 최대 30명 입장
- 초대 링크와 QR 코드 제공
- 글쓰기 화면, 그리기 화면, 결과 화면 분리
- 연필, 페인트 통, 지우개, 되돌리기, 전체 지우기
- 그리기 시간 조절

## Render에 올리는 방법

1. 이 폴더를 GitHub 저장소로 올립니다.
2. [Render](https://render.com)에서 `New` → `Web Service`를 선택합니다.
3. GitHub 저장소를 연결합니다.
4. 설정은 아래처럼 둡니다.
   - Runtime: `Node`
   - Build Command: 비워두기
   - Start Command: `npm start`
   - Health Check Path: `/healthz`
5. 배포가 끝나면 Render가 만들어준 주소로 접속합니다.

## 운영할 때 알아둘 점

현재 버전은 방 정보를 서버 메모리에 저장합니다. 그래서 무료 서버가 잠들거나 재시작되면 진행 중인 방은 사라질 수 있습니다. 실제 학교에서 오래 안정적으로 쓰려면 다음 단계에서 데이터베이스 저장을 붙이는 것이 좋습니다.

Render 무료 플랜은 접속이 없으면 잠들 수 있습니다. 수업 전에 사이트를 한 번 열어 깨워두면 편합니다.

## 로컬에서 실행

```powershell
.\start-server.ps1
```

브라우저에서 `http://localhost:4173`을 열면 됩니다.
